一个为 Minecraft 1.21.4 打造的、高度可自定义的 Fabric 聊天栏调试消息拦截与过滤模组。

---

## 🌟 核心特性 (Features)

SDR 提供了一套轻量、安全且极度可自定义的客户端聊天栏消息过滤机制，完全支持在游戏内通过图形化配置界面进行管理：

- 🚫 **多行自定义拦截规则**：
  - 允许在界面中配置多行匹配前缀（例如默认的 `[S]`），只要聊天消息开始于其中任何一个规则，就会被自动拦截展示。
- 🟢 **多行自定义白名单放行**：
  - 允许在界面中配置多行放行规则（例如默认的 `[S] [IRC]` 和 `[S] Bound`）。
  - 当拦截规则生效时，如果该消息包含（Contains）任意一条白名单子串，将直接放行，确保重要的调试或聊天消息不受干扰。
- ⚙️ **高保真 Client Settings 图形化配置界面**：
  - 游戏内按 **O** 键（可自定义）直接呼出精美的配置控制面板。
  - 完全遵循我的世界原版 UI 风格设计，包含双档功能开关、两组配有字符计数监控 (`0/8192`) 的多行编辑框，交互顺滑。
- ⌨️ **原版按键管理设置支持 (原生中英双语)**：
  - 本模组的所有控制快捷键（配置界面、快速拦截开关、快速提示开关）均已完美注册到原版我的世界“**按键绑定 (Key Binds)**”设置菜单中。
  - 玩家可以自由地将其修改为任意心仪按键，绝不与其它模组冲突。
- 💾 **无损配置持久化自动保存**：
  - 所有设置实时自动保存于 `config/southsidedebugremover.properties`。
  - 多行规则和白名单采用 Base64 加密算法存取，完美保证多行排版和特殊字符重启游戏后永不丢失。
- 🔒 **防死循环重入状态锁**：
  - 配备了基于 `ThreadLocal<Boolean>` 的拦截锁，即使开启了拦截后聊天框提示信息（`[SDR] 拦截一个调试信息，内容:`），也绝对不会引发无限递归栈溢出崩溃。

---

## ⌨️ 默认快捷键 (Default Keybindings)

本模组注册了 3 个快捷按键（均可在游戏内原版按键设置中进行重绑）：

| 快捷键 | 默认按键 | 功能描述 |
| :--- | :---: | :--- |
| **打开 SDR 配置界面** | **O** | 呼出模组的 Client Settings 配置与白名单编辑器面板 |
| **快捷切换拦截功能** | **F8** | 在游戏内快速开启或临时关闭拦截主功能（伴随金黄色美观弹窗提示） |
| **快捷切换调试提示** | **F9** | 快速开启或关闭拦截时的聊天栏 `[SDR]` 通知提示 |
